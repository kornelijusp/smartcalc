Instruction

1) Download smartcacl.exe. 2) Open terminal in directory there is smartcalc.exe. 3) Write into terminal smartcalc >function< >x< >y< (Windows) if you using LINUX you need write ./ before smartcalc. 4) Write smartcalc –help to see all mathematical function list. 5) Write smartcalc >function< –help to see specific information about function.
